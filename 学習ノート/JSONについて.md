## JSON
- JSON(JavaScript Object Notation)
- JavaScriptのオブジェクトをテキストとして表す際に使われるオブジェクトリテラルの記述方式
- JavaScriptではオブジェクトをJSONフォーマットのテキストに変換できる
- 逆にJSONテキストを元にオブジェクトを作ることもできる
